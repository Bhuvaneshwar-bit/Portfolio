To use the Sketchfab sports car model:

1. Go to: https://sketchfab.com/3d-models/generic-sport-car-low-poly-7ad3bfb2bd7449c19698fc7d0592a601

2. Download the model (you'll need to create a free Sketchfab account if you don't have one)

3. Extract the downloaded ZIP file

4. Rename the GLB file to "generic_sport_car_low_poly.glb"

5. Place the GLB file in this "models" folder
